About Upi Payment Transactions Data

The data is sourced from this Kaggle dataset.

Definitions of each variable:

  • Transaction_ID: Unique identifier for each Transaction
  • Timestamp: Timestamp of the Transaction
  • Sender_Name: Sender Name of the Transaction
  • Sender_UPI_ID: Sender UPI ID of the Transaction
  • Receiver_Name: Receiver Name of the Transaction
  • Receiver_UPI_ID: Receiver UPI ID of the Transaction
  • Amount_INR: Amount in INR
  • Status: Transaction Status
